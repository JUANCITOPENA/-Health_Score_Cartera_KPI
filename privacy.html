Política de Privacidad

Última actualización: 5 de febrero de 2026

Esta política de privacidad se aplica al objeto visual personalizado de Power BI "Health Score Cartera KPI" (en adelante, "el Visual"), desarrollado por el Ing. Juancito Peña Vizcaíno.

1. Recopilación de Datos

El Visual no recopila, almacena ni transmite ningún tipo de información personal, datos identificables o datos del negocio del usuario fuera del entorno de Microsoft Power BI.

2. Procesamiento de Datos

Todos los cálculos y visualizaciones realizados por el Visual se ejecutan localmente en el dispositivo del usuario dentro de la sesión activa de Power BI. Los datos proporcionados por el modelo de datos de Power BI se utilizan exclusivamente para renderizar los elementos gráficos (Gauge, tarjetas líquidas y etiquetas).

3. Servicios de Terceros

El Visual no realiza llamadas a APIs externas ni servicios web de terceros. No se utilizan cookies ni tecnologías de seguimiento.

4. Seguridad

El Visual cumple con los estándares de seguridad de Microsoft para objetos visuales personalizados, operando dentro del sandbox seguro proporcionado por Power BI.
