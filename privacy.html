Política de Privacidad

Última actualización: 5 de febrero de 2026

Esta política de privacidad describe cómo se manejan los datos en el objeto visual personalizado de Power BI "Health Score Cartera KPI", desarrollado por el Juancito Peña Vizcaíno.

Resumen de Privacidad: Nuestro visual está diseñado bajo el principio de "Privacidad por Defecto". No recolectamos datos fuera de su infraestructura de Microsoft.

1. Recopilación de Datos

El Visual no recopila, almacena ni transmite ningún tipo de información personal, datos identificables o datos del negocio del usuario fuera del entorno seguro de Microsoft Power BI.

2. Procesamiento de Datos

Todos los cálculos realizados por el Visual se ejecutan estrictamente en la memoria local del dispositivo del usuario. El acceso a los datos del modelo se utiliza exclusivamente para la representación gráfica de:

  • Indicadores Gauge de rendimiento.
  • Tarjetas de estado de llenado líquido.
  • Etiquetas dinámicas de KPIS.

3. Servicios de Terceros

El Visual es una solución aislada. No realiza llamadas a APIs externas, no utiliza cookies de seguimiento y no se conecta con servicios web de terceros.

4. Estándares de Seguridad

Esta solución cumple con los requisitos de certificación de Microsoft AppSource, operando dentro del sandbox seguro (aislamiento) que proporciona Power BI, garantizando que el visual no tenga acceso a recursos del sistema o de red no autorizados.

5. Contacto y Transparencia

Para cualquier consulta técnica o aclaración sobre esta política, puede dirigirse a los canales oficiales:
